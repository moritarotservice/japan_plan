## 第一輪建議優先順序

1. Salon Guest Check：日本美容 / 沙龍外國客預約前確認工具
2. 外國旅客 FAQ 產生器
3. 多語入住前訊息產生器

## 目前策略

先做網站與小工具驗證，不先做完整 SaaS。

每個小工具網站的目標是測：

- 是否有人點擊廣告
- 是否有人完成表單
- 是否有人輸入真實資料
- 是否有人留下 Email / LINE
- 是否有人點擊預約 demo 或導入 CTA
- 是否有人願意付 setup fee
